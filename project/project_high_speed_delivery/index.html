Summary

  • Researched at http://ishikawa-vision.org/fusion/UAVdelivery/index-e.html

Although research on physical distribution using unmanned aerial vehicles (UAVs) has seen increasingly significant interest, the task of automatically loading a parcel onto a UAV has not been researched adequately. In this study, to design an automatic UAV delivery system, we achieved the task of non-stop handover of a parcel to an airborne UAV. For the handover task, we developed a novel tracking system with high-speed, multi-camera vision using cameras with different frame rates. Through sensor fusion, the proposed system overcomes the problem associated with tracking far away objects by using high-speed cameras with short exposure times. The proposed system demonstrates that it is feasible to combine both high-speed object tracking (1000 fps) and distant object tracking.

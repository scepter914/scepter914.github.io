Setting Velocity on Joints and Links for Gazebo ver9.6

概要

  • Setting Velocity on Joints and Links (Gazebo ver7.0)をGazebo version 9.6 で動くようにした
  • https://github.com/scepter914/set_vel_plugin Github

やったこと

流石にROSを少し触った方がいいんじゃないかと思い， 「勉強がてらGazeboサンプル動かすかー」と思ってやってみたら， まずサンプルはversion古いやつやでっていう罠に引っかかった．

Ubuntu18.04LTSでやっていてversion下げるのすげー面倒そうだしデバッグした． デバッグでやったこと，参考リンクはREADMEに書いたので，そちらをご参照を．

余談ですが，新年色々触って遊んだのでblogの更新が捗りますね．

Environment

  • Ubuntu18.04LTS
  • ROS: Melodic
  • Gazebo 9.6

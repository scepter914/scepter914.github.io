遭遇したエラー

ページのリンクが崩壊する

  • lenient-tag-lookupオプションを起動する
 gollum --config config.rb --lenient-tag-lookup &

レイアウトが崩れる

  • 主にsidebarが変な挙動する
    • 幅が狭くなって左側に寄る減少が起こる
  • css customをonにしているとなる
  • 結論としてブラウザのキャッシュが悪さをしていた
    • chromeならctrl-f5をすればよい

sudo gem install gollumが通らない

  • wsl on windowsで起こった
  • cmakeが通らない
  • sudo apt install openssl-devel

config fileの対応

  • https://github.com/gollum/gollum/wiki/5.0-release-notes
  • 一部無くなった機能があったりしたのでconfig系を修正
    • css
    • config,rb
  • docker対応
    • rubyのversionを上げる必要があった
    • buildし直してdocker hubへ登録した
